# damonvox

This website was created by **Damon Vox** for **Damon Vox**. It serves as a personal portfolio showcasing his work and creative vision.

## Development

**Prerequisites:** Node.js

1.  **Install dependencies:**
    ```bash
    npm install
    ```
2.  **Environment Setup:**
    Set the `GEMINI_API_KEY` in `.env.local` to your Gemini API key.
3.  **Run locally:**
    ```bash
    npm run dev
    ```
4.  **Build for GitHub Pages:**
    ```bash
    npm run build
    ```
    The build output is configured for the `/docs` directory to support GitHub Pages deployment.
